# Project Description

This repository contains a Catholic archive and public-history indexing workspace for the St. Bonaventure reliquary project. It is designed to keep visible-label transcription, saint/entity registration, image and crop manifests, provenance controls, uncertainty flags, devotional tradition, and historically verified claims separate and traceable.

The project is an active archive build and public-history methods candidate.

## Short description

Catholic archive and public-history indexing project for St. Bonaventure reliquary provenance, transcription, entities, and uncertainty controls.

## Scope boundary

Do not collapse devotional tradition, historical interpretation, and evidentiary claims into one category. Keep label, image, crop, entity, and uncertainty records traceable.
