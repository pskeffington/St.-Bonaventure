# Status and Roadmap

## Current status

**Active archive build.**

This repository supports Catholic archive and public-history indexing work for the St. Bonaventure reliquary project. It should be treated as a provenance-controlled archive workflow that separates visible-label transcription, entity modeling, image/crop tooling, devotional tradition, and historically verified claims.

## Public-facing status language

Use the following wording in CV, portfolio, and public research-status outputs:

> St. Bonaventure archive and public-history indexing project using reliquary section mapping, visible-label transcription, entity modeling, crop tooling, and provenance controls; uncertainty flags and historical/devotional claim separation remain active work.

## Near-term roadmap

1. Complete the section index.
2. Maintain visible-label transcription separately from interpretation.
3. Add or refresh the image-crop manifest.
4. Add uncertainty flags for unresolved identifications.
5. Separate devotional tradition from verified historical evidence.
6. Prepare a public-history methods note only after provenance controls are stable.

## Validation gates

- Do not collapse devotional, historical, and evidentiary claims into one category.
- Keep image, crop, label, and entity records traceable.
- Mark uncertain identifications explicitly.
