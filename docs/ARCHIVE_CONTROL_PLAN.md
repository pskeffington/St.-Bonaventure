# Archive Control Plan

## Purpose

Define the archive-control structure for St. Bonaventure reliquary indexing.

## Current status

Active archive build.

## Control objects

| Object | Status | Notes |
|---|---|---|
| Section index | Pending update | Map sections and physical layout |
| Visible-label transcription | Active | Keep separate from interpretation |
| Saint/entity register | Active | Maintain uncertainty flags |
| Image/crop manifest | Pending update | Link crop IDs to source images |
| Provenance notes | Active | Track source and evidence status |
| Devotional vs historical claim register | Pending | Separate tradition from verification |

## Rules

1. Transcribe visible labels before interpretation.
2. Keep uncertain identifications flagged.
3. Link crops to source image records.
4. Separate devotional tradition, historical interpretation, and evidentiary confirmation.
5. Preserve source and provenance notes for every public claim.
