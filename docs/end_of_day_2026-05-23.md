## Photographer assignment note

Future publication-grade photography should capture each object in controlled conditions rather than relying on wall-case photos alone.

Minimum photo set per relic/object:

- full object on white or neutral background
- close front image of the theca
- close image of the theca inscription
- reverse/back image if seals, threads, or closures are visible
- scale-card or ruler image
- color-reference card image per session if available

Recommended file pattern:

```text
REL-0001_full.jpg
REL-0001_theca_front.jpg
REL-0001_theca_label.jpg
REL-0001_scale.jpg
REL-0001_reverse.jpg
```
